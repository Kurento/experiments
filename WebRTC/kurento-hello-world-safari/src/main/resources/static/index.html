Kurento - Safari compatibility test

This web application shows a WebRtcEndpoint connected to itself (loopback). Take a look to the Media Pipeline.

To run this demo follow these steps:

  1. Open this page with a WebRTC-compliant browser (Chrome, Firefox, Safari).
  2. Click on Start to begin the stream negotiation.
  3. Grant access to the camera and microphone.
  4. Click on Stop to finish the test.
